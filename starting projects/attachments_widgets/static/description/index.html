Attachments widgets

What does this module solve?

Default behavior

By default, in odoo, a field of type many2one or binary on the ir.attachment table has the function just upload and download.

Behavior after install

After installing this module, widgets will be available, at the moment only one is an audio/video widget.

When using which all audio/video files will be displayed in a media player with the ability to play and download, and in editing mode

it is still available to download a new file, this is very convenient when working with a large number of audio/video files,

such as recordings of telephone conversations, for example

Features

  • Audio 2 widgets for many2one/binary attachment
  • Video 2 widgets for many2one/binary attachment (with support width option)
